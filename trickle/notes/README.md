# BSEDC Website Redesign

This project is a modern UI/UX redesign of the Bihar State Electronics Development Corporation Ltd. (BSEDC) official portal, focusing on achieving an eGovernance award-winning aesthetic.

## Tech Stack
- React 18
- TailwindCSS
- Lucide Icons

## Multi-Page Architecture (MPA)
- `index.html`: Home page
- `about.html`: About Us page (Mission & Vision)
- `board.html`: Board of Directors page
- `md-list.html`: List of Managing Directors page
- `team.html`: Our Team page
- `services.html`: Services page
- `projects.html`: Projects page
- `news.html`: News & Events page
- `circulars.html`: Letters & Circulars page
- `tenders.html`: Tenders page
- `contact.html`: Contact Us page
Each page has its corresponding `[page]-app.js` entry file.

## Maintenance Rules
- Whenever major layout changes or new pages are added, update this README to reflect the latest site architecture.
- Follow the enterprise color theme established in `index.html`.
